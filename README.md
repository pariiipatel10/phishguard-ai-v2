# PhishGuard AI 🛡️

PhishGuard AI is a modern cybersecurity awareness and phishing detection platform designed to help users identify suspicious emails, malicious URLs, and learn cybersecurity concepts through an interactive AI assistant.

---

## 🚀 Features

### ✅ Email Phishing Scanner

* Analyze suspicious emails
* Detect phishing indicators
* Risk percentage analysis
* Dynamic threat levels

### ✅ URL Scanner

* Detect malicious and suspicious URLs
* High / Medium / Low risk analysis
* Cybersecurity-focused UI

### ✅ AI Cybersecurity Assistant

* Interactive chatbot
* Cybersecurity Q&A
* Static intelligent responses
* Covers phishing, malware, VPN, ransomware, firewalls, DDoS, encryption, and more

### ✅ Authentication System

* User Signup
* User Login
* Protected dashboard routes
* Logout functionality
* LocalStorage-based session management

### ✅ Dynamic Dashboard

* Threat analytics
* Interactive pie chart
* Dynamic risk statistics
* Modern cyber UI

### ✅ Responsive Cybersecurity UI

* Neon cyber theme
* Hover animations
* Toast notifications
* Responsive design
* Interactive navigation

---
